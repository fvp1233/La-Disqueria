# Repositorio de Tienda en linea La disqueria

La aplicación web esta creado con React y esta estilizada con Tailwind y el framework de shadCn

## Para ejecutar la aplicación seguir los siguientes pasos:

1- Clonar el repositorio
2- Instala las dependencias dentro de ambas carpetas la-Disqueria y la-Disqueria-Admin con el comando npm i dentro de cada uno
3- Ingresar a la carpeta del proyecto al cual queremos entrar con el comando cd y el nombre de la carpeta
4- Correr el programa con el comando npm run dev

## Dependencias del proyecto

Dependencias para el funcionamiento de la app, si creas un proyecto nuevo, deberas de instalar las siguientes dependencias en el proyecto.

```bash
npm install tailwindcss @tailwindcss/vite
npm install -D @types/node
npx shadcn@latest init
npm install clsx tailwind-merge

npx shadcn@latest add dropdown-menu
npx shadcn@latest add button
npx shadcn@latest add card
